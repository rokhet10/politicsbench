# Archived framing pilot (pro / anti)

Snapshot of the **framing** experiment: paired variants differed in **speaker order, primacy, and rhetorical lead** while keeping similar facts.

- `scenario_prompts_pilot_framing.txt` — full prompt file (`N-pro` / `N-anti`).
- `manifest_framing.json` — manifest for that file.

The wording-only experiment (`N-wa` / `N-wb`) uses the same beats and order of presentation; **surface wording** differs (synonyms, sentence shape, concision).
